# RoyLM-1

RoyLM-1 is the next iteration of [RoyLM-0](https://github.com/ryouol/RoyLM-0),
a small character-level GPT trained from scratch. Like its predecessor, it is a
readable transformer-from-scratch project: the model learns to predict the next
character, then generates text by repeating that prediction step.

It is not a chatbot or instruction-following model. It continues text in the
style of the training data.

> This repo is a fresh starting point. Architecture, training setup, and
> benchmarks will be filled in as the model takes shape.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
